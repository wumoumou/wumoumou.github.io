php和python实现跨域共享session

发表于 2018-06-26 | 分类于 python | 阅读次数

用python写了一个web工具，为了放至php的后台但只调用一套身份验证机制，于是研究了一下跨域的session共享

一、实现方式

php在执行session_start()函数后会在配置的session_path目录下生成一个以sessionId命名的文件，文件内容即是$_SESSION变量的内容。浏览器端会生成一个以PHPSESSID命名的cookie，cookie的内容就是sessionId。所以根据这个可以通过传递sessionId的方式，实现同一服务器跨域共享session。
